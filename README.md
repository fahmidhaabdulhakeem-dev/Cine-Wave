# 🎬 Cine-Wave
A sleek, responsive, and keyless Movie & TV Show Discovery Web Application built with HTML, Vanilla CSS, and JavaScript. Powered by the [TVMaze API](https://www.tvmaze.com/api).
---
## 🌟 Key Features
- **🔍 Smart Live Search**: Type queries to search for shows instantly. Uses a debounced input mechanism to optimize API requests and prevent rate-limiting.
- **✨ Dual Premium Themes**: Switch seamlessly between **Midnight Dark** (default) and **Cyber Neon** modes. Your preference is persisted via local storage.
- **🎨 Glassmorphic Responsive UI**: A mobile-first, grid-based interface with soft glassmorphic panels, glow overlays, hover scaling, and clean micro-animations.
- **⚡ Skeleton Loaders (Shimmers)**: Shows skeleton card layouts during API calls to ensure a smooth, fluid user experience with no layout shifts.
- **🏷️ Quick Genre Filter tags**: Instantly filter search results by genres (Action, Drama, Comedy, Sci-Fi, Thriller, etc.).
- **⭐ Rating Range Filter**: Filter cards dynamically using a custom range slider based on user-rating scores.
- **🔄 Multi-criteria Sorting**: Sort results by Relevance/Popularity, Ratings (High/Low), Alphabetical Name (A-Z/Z-A), or Release Date (Newest/Oldest).
- **📋 Extended Details Modal**: Click any card to see full summaries, network platforms, schedule times, runtime, language, and key cast members (featuring actors' headshots).
---
## 🛠️ Technology Stack
- **Structure**: Semantic HTML5 markup (SEO-optimized with meta headers, descriptions, and viewport properties).
- **Style**: Custom Vanilla CSS3 (utilizing CSS variables, CSS grid, Flexbox, custom range sliders, custom scrollbars, and keyframe animations).
- **Logic**: Vanilla ES6 JavaScript (debounced event handlers, Fetch API, dynamic DOM rendering, and LocalStorage manipulation).
- **Icons**: FontAwesome v6.4 (linked via CDN).
- **Typography**: Google Fonts ("Outfit" & "Plus Jakarta Sans")
